 블록 퍼즐(가칭) 프로젝트 상세 요구사항## 목적Canvas API 기반 게임 제작: DOM 요소 생성 오버헤드 없이 부드러운 프레임과 그래픽 효과를 구현.순수 자바스크립트(Vanilla JS) 활용: 라이브러리 의존성을 줄여 핵심 알고리즘(충돌 체크, 라인 클리어) 이해도 극대화.## 필수 기능 (확정)Canvas 렌더링 시스템:$10 \times 10$ 메인 보드와 하단 블록 대기소(3칸)를 Canvas 위에 드로잉.드래그 앤 드롭 (Custom 구현):mousedown, mousemove, mouseup 이벤트를 활용해 블록 이동 구현.블록을 놓을 때 격자 중심점에 맞게 스냅(Snap)되는 기능.다양한 블록 패턴:테트리스 기본형 외에 $1 \times 1, 3 \times 3, 1 \times 5, L$자 형태 등 블록 블라스트 특유의 다양한 패턴(약 15종 이상) 적용.라인 클리어 및 점수:가로/세로 완성 시 즉시 제거.콤보 시스템: 연속으로 줄을 제거할 경우 가산점 부여.로컬 데이터 관리:localStorage를 사용하여 최고 점수(High Score) 영구 저장.## 디자인 요구Dark Mode 테마: 어두운 배경에 채도가 높은 네온 컬러 블록으로 가독성 확보.고스트 블록(Ghost Block): 블록을 드래그하여 보드 위에 올렸을 때, 배치될 위치를 반투명하게 미리 보여주는 가이드 기능.애니메이션: 줄이 사라질 때 파티클 효과 또는 페이드아웃 효과 적용.## 기술 스택언어: HTML5, CSS3, JavaScript (ES6+)그래픽: Canvas API (2D Context)서버: 로컬 개발 서버 (Live Server)저장소: Browser LocalStorage
